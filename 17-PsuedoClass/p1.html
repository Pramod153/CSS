CSS PSEUDO-CLASSES

The Css pseudo-classes allow you to style the dynamic states of an element such as hover, active and focus state, as well as elements that are existing in the document tree but cant be targeted via the use of selectors without adding any ID's or classes to them, for example, targeting the first or last child elements.

A pseudo-class starts with a colon(:). Its syntax can be given with:

selector:pseudo-class {property:value;}
