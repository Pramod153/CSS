Defining color values

color in css most often specifies in following formats

  1. color keyword-link red,green,blue,orange and etc
  2. Hexadecimal (2^8=256) value like #000000,#ffffff and etc
  3. RGB(RED,GREEN,BLUE) value like- rgb(255,0,0),rgb(255,255,255),rgb(0,0,0)
  4. hsl(hue,saturation,lightness)values
  5. hsla(hue,saturation,lightness,alpha)values
